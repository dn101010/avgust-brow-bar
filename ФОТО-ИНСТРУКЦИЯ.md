# Как добавить фото на сайт

## Шаг 1: Подготовь фото
- Формат: JPG
- Размер: минимум 800px по ширине
- Ориентация: горизонтальная (альбомная) для услуг, вертикальная (портретная) для мастеров
- Качество: хорошее освещение, без фильтров

## Шаг 2: Назови файлы
Каждый файл должен называться строго так:

### Фото услуг (public/images/services/)
- ukladka.jpg - Долговременная укладка
- arhitektura.jpg - Архитектура бровей
- okrashivanie.jpg - Окрашивание бровей
- laminirovanie.jpg - Ламинирование ресниц
- muzhskoe.jpg - Мужское оформление
- osvetlenie.jpg - Осветление бровей
- kombo.jpg - Комбо брови + ресницы
- narashivanie.jpg - Наращивание ресниц
- korrekciya.jpg - Коррекция наращивания
- snyatie.jpg - Снятие наращивания

### Фото мастеров (public/images/team/)
- tatyana.jpg - Фото Татьяны
- mariya.jpg - Фото Марии

### Фото атмосферы (public/images/atmosphere/)
- kofe.jpg - Кофе в студии
- sladosti.jpg - Сладости
- kushetka.jpg - Кушетка
- studio.jpg - Пространство студии

### Фото работ (public/images/portfolio/)
- work-01.jpg ... work-12.jpg - Фото работ (до/после)
